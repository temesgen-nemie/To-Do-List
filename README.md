## Features
- [x] Show tasks list
- [x] Add new task
- [ ] Complete task
- [x] Delete task
- [x] Edit task
- [x] Clear all tasks
- [ ] Clear complete tasks
- [x] Save on local storage
- [x] Reverse sort by id

- [x] Custom scrollbar
- [x] Responsivity
